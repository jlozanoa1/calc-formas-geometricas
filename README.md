# Calculadora de formas geométricas
Calculadora simple de línea de comandos para obtener el área de diferentes figuras geométricas.

## Funcionalidades
Las funcionalidades que presenta esta aplicación son las siguientes.
- Área de un cuadrado ([**característica/área-cuadrado**](https://github.com/jlozanoa1/calc-formas-geometricas/tree/caracter%C3%ADstica/%C3%A1rea-cuadrado)): Calcular el área de un cuadrado, realizado por Luis Maza.
- Área de un rectángulo ([**feature/area-rectangulo**](https://github.com/jlozanoa1/calc-formas-geometricas/tree/feature/area-rectangulo)): Calcular el área de un rectángulo, realizado por Juan Rico.
- Área de un triángulo ([**feature/area-triangulo**](https://github.com/jlozanoa1/calc-formas-geometricas/tree/feature/area-triangulo)): Calcular el área de un triángulo, realizado por Juan Lozano.
- Área de un círculo ([**característica-área-círculo**](https://github.com/jlozanoa1/calc-formas-geometricas/tree/caracter%C3%ADstica-%C3%A1rea-circulo)): Calcular el área de un círculo, realizado por Sebastián Hernández.

## Cómo ejecutar el programa
Para ejecutar el programa (asumiendo que ya tiene JDK instalado), debe situarse en el directorio src, luego compilar el fichero *Main.java* con el comando **javac** y proceder a ejecutar *Main.java* con el comando **java**.
```bash
cd src
javac Main.java
java Main
```
O por otro lado, si prefiere, puede utilizar un IDE y ejecutar el fichero **Main.java** directamente.
